**CraftLegacyLauncher** is a simple launcher for the legacy console-style block game port created by **SmartCMD**.
The **Launcher** makes it easier to start the game, manage settings, and connect to LAN without needing to type long commands to CMD.

## Features

- A simple UI
- LAN input
- Username Input
- Lightweight python file

## Installation

1. Download(https://github.com/PieD3V/CraftLegacyLauncher/releases/tag/Debug) files as a .ZIP
2. Extract .ZIP
3. Run the file main.py and press play!
4. It will download the latest EXE build from **SmartCMD's** github repo.
5. When its done, enjoy!

   **Help**
6. If it wont launch add path to minecraft_install folder that will be made if you download the latest version in the launcher.
7. To find were to set path goto settings.

## Requirements

- Windows 10-11
- **Radmin** VPN for LAN between Computers.
